## 배열 (Array)
    1. 데이터를 나열하고, 각 데이터를 인덱스에 대응하도록 구성한 데이터 구조
    2. 파이썬에서는 리스트 타입이 배열 기능을 제공하고 있음

### 배열을 사용하는 이유 
    1. 같은 종류의 데이터를 효율적으로 관리하기 위해 사용
    2. 같은 종류의 데이터를 순차적으로 저장

### 장점
    1. 빠른 접근 가능 

### 단점
    1. 배열의 크기를 미리 선언해야한다. (파이썬은 거의 ArrayList랑 비슷해서 크게 상관없다)
    2. 추가 / 삭제가 쉽지 않다.
